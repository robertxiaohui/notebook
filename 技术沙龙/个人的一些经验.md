# 个人的一些经验

## 考研or找工作

```
其实就我目前找工作的经验来讲，其实考研和找工作是可以一起进行的(先尝试考研，考不上在找工作)，这样的话需要平时应用技术的积累，比如参加一些实习，平时做一些java的项目
因为我舍友也进行了考研，他报考的是四川大学的计算机专业，今年也没考上，今年计算机考研是非常的内卷，考试人数分数也都很高，他平时实习，项目做的也少，相对应用能力就比较差，如果这时候去找工作的话应该会碰壁


1. 考研的学习是有助于找工作的，对于一些中上的公司，在面试的时候会倾向于考察学生的基础，会大量询问数据结构的知识，也会涉及计算机网络大部分是tcp/udp协议 http协议，也会询问一些操作系统相关的知识(计算机组成原理我目前是没有被问到过)
所以如果是进行先考研，考不上之后再找工作其实也没关系，因为考研会大大强化对数据结构，操作系统等课程的理解，自己的基础也会更加扎实

2.大学的个人规划是很关键的，机会都是留给有准备的人，这句话我现在非常能体会到，大家都说考研要早点准备，其实找工作也更是如此，如果想进大厂大三上就需要开始实习，开始准备大厂的笔试面试等，所以大二结束的时候就要明确自己是要考研还是找工作，如果认真考研，考不上再找工作基础扎实进个中上厂应该没有问题，但是进大厂的过程就和考研一样都是提前准备

3. 虽说考研和工作可以同时进行，但考研也不能盲目，因为目前的趋势考研形式只会越来越严峻，很多人考研可能只是跟风，或者对工作的迷茫，而且大部分人考研的最终目的也是找一份好工作

我个人平时计算机的各种知识也学，学的比较杂，但也比较浅显，大三上因为疫情也在家，大三下准备去找实习，发现自己框架啥的不会发现好像挺难找实习的(其实现在来看这是一个误区，可能那时候投的公司都是比较小的公司，小公司就注重你的上岗后的上手能力，需要你一进去都会，其实这样的小公司去了发展前景也不大，没有完善的培养方案)，后面就自己去网上学，但前期学习是比较累的，后面找了一个学长带我学，学长技术很强人也非常不错，跟着他学到了蛮多东西。
其实在没学框架以前觉得框架是一个比较高大上的东西，学习起来比较难，内心上就有点惧怕吧
其实学的时候发现框架本质就是其他人开发的一些java的类，我们大部分学习框架就是去如何使用他，这个难度就很简单，并且跟你java基础部分的学习情况其实没有多大关系
当然如果要自己开发一些框架，那么java基础得学习的非常不错

自己学的一个问题就是可能学的不够系统，所以找人带非常有必要，不要怕学长学姐不愿意帮你，其实很多学长学姐都非常乐意帮助学弟学妹的

现在学习完之后去面试，工作也面试了挺多家，发现进一个中小厂是非常容易的一件事，这时候回想一下就挺后悔为什么没有早点做准备，如果早做准备说不定也能争取一下大厂的机会






```

## 关于上培训班的理解

```
其实上培训班本质上就是一个花钱买时间，花钱买自律，寻找一个引导者的过程，如果从这个角度来讲我认为是非常值得的
虽然现在网络上面编程的学习视频非常多，但是一个人自学还是会走非常多的歪路，走歪路就会浪费非常多的时间，我个人高考完后就看着网上的java基础视频进行了学习，后面在学校也陆续又看了好了2遍java的基础学习视频，但是也一直就停留在看基础的情况下，想着后续的学习肯定要扎实的java基础才能进行，一个小白听到框架那时候觉得非常高大上好像离自己很遥远，就一直觉得自己基础不行往后面学应该理解不了，这其实就是走了歪路，真正编程中使用的框架反而比java基础要简单，学习框架的使用更多的就是遵循人家设定的规则，依样画葫芦，当然还是那一点要自己开发框架java基础要学好，要在java这条路上走的远肯定也是想着自己能够开发出一些框架让大家都能使用的，这当然是一个比较高的目标


自学还有一点就是自己能不能有那一份自律能够认真学习下去，很多时候可能就是兴趣突然来了学了一下，不能长久的坚持


不过我也找工作前也参加了一个月的就业快速培训，里面是加强一些框架的使用，面试的一些问题的讲解，深入理解一个个人编写的项目，要能够完全了解项目的难点然后如何解决这个难点，参加完培训后个人感觉是非常有帮助的，就好比原先是一个人独自摸索，现在有老师引导你，感觉前面的路就走的挺顺利的，确实面试的时候也都是问一些类似的问题，自然能轻松应对

前几天和这边图灵培训的老师交流了一下，也查看了图灵培训的教学，因为图灵培训他们的招生目标都是面向南邮，河海这样的优质大学，优质大学的生源学习能力都能有保证，所以他们的培养方案中不仅包括了传统的java基础到企业级web开发，还包括了408考研，说实话我还蛮后悔为啥仔细比较一下培训中心就去上了，如果我来的是图灵因为包含408考研我可以先去考研尝试一下再去找工作，当然因为生源优秀，要教授优质的生源那么老师的水平也必须要高，图灵的老师都是华为等大厂的高级工程师，优质的生源优质的老师，最后培训出来的结果当然也非常不错，图灵和大厂都有合作有内推名额，学习优异的可以去到大厂（这当然是每个程序员的目标）

最后说一个，如果想培训这个时间一定要乘早，身边也有挺多考研失败之后去参加培训的，但是等他们培训完企业招人的黄金期就过了，那也只能等下一年，这个时间成本也是要考虑的，就我个人而言参加培训班等于是拥有了培训班的一些资源，从长远角度看培训班的钱花费是值得的

```
